# Inequa

Inequa — нативный Android‑калькулятор для решения алгебраических неравенств с пошаговым объяснением.

## Что уже умеет

- Компактный калькуляторный интерфейс на Kotlin + Jetpack Compose + Material 3.
- Встроенная математическая клавиатура (`0-9`, `x`, `y`, `z`, `(`, `)`, `+`, `-`, `/`, `^`, `²`, `³`, `>`, `<`, `≥`, `≤`, `,`, `|`, `√`, `C`, `⌫`, `=`).
- Стабильный ввод и удаление токенов (`>=`, `<=`, `sqrt(`, `^2`, `^3` и др.).
- Нормализация ввода:
  - поддержка Unicode и ASCII операторов (`≥`/`≤` и `>=`/`<=`);
  - обработка математического минуса `−`;
  - поддержка `√(...)` и `sqrt(...)`;
  - поддержка `|...|` и `abs(...)`;
  - поддержка переменных `x`, `y`, `z`;
  - поддержка степеней в виде `^n` и верхних индексов.
- Автоформатирование выражения для удобного чтения.
- Краткий ответ на главном экране + подробные шаги в отдельной нижней панели.
- График решения на числовой прямой для интервального ответа.

## Поддерживаемые типы неравенств

### 1. Линейные
Примеры:

- `2x + 3 > 7`
- `-3x + 12 >= 0`
- `0,5x - 1 < 4`

### 2. Квадратные
Примеры:

- `x² - 5x + 6 <= 0`
- `y² - 1 >= 0`

### 3. Дробно-рациональные (базовые формы)
Примеры:

- `(2x - 1) / (x + 3) > 0`
- `(y + 2) / (y - 5) <= 0`

### 4. С модулем (базовые формы)
Примеры:

- `|x - 3| >= 2`
- `abs(2x + 1) < 5`

### 5. С корнем (базовые формы)
Примеры:

- `√(x + 1) > 2`
- `sqrt(2x - 3) <= 5`

## Что пока не реализовано полностью

- Степени выше второй
- Дробно-рациональные неравенства в полностью общем виде
- Составные и системы

Архитектура уже подготовлена для расширения этих случаев.

## Архитектура

Проект разделён на слои:

- `ui` — экраны, компоненты, тема, `ViewModel`, `UiState`
- `domain` — модели, нормализация, форматирование, парсинг, решатели
- `data` — зарезервирован под историю, теорию, сохранение состояния

Ключевые файлы:

- `MainActivity` — только запуск приложения
- `InequaApp` — корневая точка Compose
- `SolverScreen` — главный калькуляторный экран
- `SolverViewModel` — управление состоянием и действиями ввода/решения

## Сборка и запуск

### Требования

- JDK 11+
- Android SDK (compileSdk 36)
- minSdk 26

### Debug APK

```bash
./gradlew :app:assembleDebug
```

Windows:

```powershell
gradlew.bat :app:assembleDebug
```

### Release APK

```bash
./gradlew :app:assembleRelease
```

Windows:

```powershell
gradlew.bat :app:assembleRelease
```

APK после сборки:

- `app/build/outputs/apk/debug/app-debug.apk`
- `app/build/outputs/apk/release/app-release.apk`

## Тесты

```bash
./gradlew :app:testDebugUnitTest
```

## Важно для публикации

Сейчас `release` конфигурация подписывается debug-ключом (для удобства локальной проверки).
Для публикации в Google Play необходимо заменить подпись на production keystore.
